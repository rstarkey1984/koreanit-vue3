# Vue 기반 뉴스 리더 프로젝트

## 프로젝트 개요

이 프로젝트는 **Vue 3를 중심으로 한 프론트엔드 뉴스 리더 애플리케이션**이다.  
Google News RSS 데이터를 활용하여 최신 뉴스를 조회하고,  
읽음 표시 및 즐겨찾기 기능을 제공한다.

본 프로젝트의 목적은 **Vue 프레임워크의 역할과 구조를 이해하고**,  
CDN 방식이 아닌 **실무형 Vue 프로젝트 흐름을 경험하는 것**이다.

---

## 프로젝트 목표

- Vue를 사용한 상태(State) 기반 UI 구조 이해
- DOM 직접 제어 방식과 프레임워크 방식의 차이 체감
- 컴포넌트 단위 UI 설계 사고 방식 학습
- 프론트엔드 프로젝트의 기본 구조와 역할 분리 이해

---

## 핵심 개념

- 화면은 DOM이 아니라 **상태(State)를 기준으로 구성된다**
- Vue는 DOM을 직접 다루지 않고 **상태와 화면의 관계를 관리한다**
- 컴포넌트는 화면을 구성하는 최소 단위다
- 프로젝트는 단일 파일이 아닌 **역할별 구조**로 관리된다

---

## 프로젝트 구성 개요

- Vue 3 기반 컴포넌트 구조
- 화면 단위(View)와 재사용 컴포넌트 분리
- 데이터 처리 로직과 UI 로직 분리
- 상태 변경에 따른 자동 UI 갱신

---

## 주요 기능 범위

- 뉴스 목록 표시
- 읽음 상태 관리
- 즐겨찾기 관리
- 상태 기반 화면 렌더링

---

## 프로젝트 성격

이 프로젝트는 단순 기능 구현이 아닌  
**프레임워크가 왜 필요한지 이해하기 위한 학습용 실무 프로젝트**다.

Vue 문법 암기보다  
**UI 흐름과 구조를 설계하는 사고 방식**에 초점을 둔다.

---

## 정리

> CDN Vue는 개념을 이해하기 위한 단계이고,  
> Vue + 빌드 도구 기반 프로젝트는 실제 서비스를 만들기 위한 구조다.
